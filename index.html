A Laravel-based API for a dice game where players roll two dice each round. Winning occurs when the dice sum equals 7.

  • ✏️ Ceate, Edit, Delete Player
  • 🎲 Roll Dice, Delete Player Games
  • 📊 Player Stats, Games
  • 🏆 Ranking, Win Rates
  • 🧑 Incloudes roles
  • 🚀 MySQL DB, Laravel Passport for auth, admin/player roles, and automated deployment.
